Module 1 Syllabus

Module 1 will cover an introduction to programming in the respective language of the cohort class, Java or C#. Included in the material will be Data Types, Expressions and Statements, Object Oriented Programming Principles, Exception Handling, File I/O, and Testing using TDD.
